## 🐛 常见问题

### Q1: 看不到背景动画？
**可能原因**：
- Canvas 渲染问题
- 浏览器不支持

**解决**：
1. 打开浏览器控制台（F12），查看错误
2. 尝试刷新页面（Ctrl + F5）
3. 使用 Chrome 或 Edge 浏览器

### Q2: 主题切换不生效？
**可能原因**：
- CSS 变量未更新
- 浏览器缓存

**解决**：
1. 硬刷新（Ctrl + Shift + R）
2. 清除浏览器缓存

### Q3: 拼音显示不正确？
**说明**：
- 拼音映射只包含常见姓氏和字
- 不认识的字会原样显示

**扩展拼音库**：
编辑 `src/lib/utils.ts`，在 `PINYIN_MAP` 中添加更多字。

### Q4: 跳转后 404？
**原因**：
- campus 页面未创建

**解决**：
确保已复制 `src/app/campus/page.tsx` 文件。

### Q5: TypeScript 报错？
**解决**：
```powershell
# 在 IDEA 中重启 TypeScript 服务
# 或运行：
pnpm db:generate  # 重新生成 Prisma 客户端
```
